How To Draw a Smiley Face


Step 1 - Draw a circle
Step 2 - Draw 2 dots inside the circle that are next to each other
Center: arc(100, 75, 50, 0 * Math.PI, 1.5 * Math.PI) Start angle: arc(100, 75, 50, 0, 1.5 * Math.PI) End angle: arc(100, 75, 50, 0 * Math.PI, 1.5 * Math.PI) Step 3- Draw a u-shaped line underneath the 2 dots - still insiide the circle Step 4 - There is no step 4, you've already completed drawing the circle
